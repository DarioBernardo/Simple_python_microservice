Documentation on how this microservice works:

The mircoservice has two end points:

  • /add
  • /visualize

/ADD

Example of how to add a data poing to account number 123:

http://localhost/add?account_no=123&date=2017-01-01&ledger_balance=10&cleared_balance=10
TRY HERE

/VISUALIZE

It is possible to visualize the balance historical data for an account number with this link format:

http://localhost/visualize?account_no=123
